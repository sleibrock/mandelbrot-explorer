Mandelbrot Explorer
---

Mandelbrot Explorer is a web-based Mandelbrot fractal generator. The goal is to provide exploration capabilities by zooming in out select areas with a rectangular selection.

The code is written in Zig and compiled to WASM. The WASM library is loaded by JavaScript into a web page, where you can see a Mandelbrot fractal for yourself. Additional features coming like changing the theme of the fractal output and selecting Julia fractal equations instead of Mandelbrot.
